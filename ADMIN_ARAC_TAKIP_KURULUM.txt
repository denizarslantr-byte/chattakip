TNB canlı bağlantı notu:
Statik GitHub Pages doğrudan TNB SOAP servisine bağlanamayabilir. Canlı bağlantı için Firebase Function proxy URL alanı doldurulmalıdır. Proxy yokken sistem Demo Mod ile açılır; araç/otel/plaka verilerini Firebase'e kaydedebilirsiniz.
